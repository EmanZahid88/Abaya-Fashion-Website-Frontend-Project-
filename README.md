# 🧕 Abaya Fashion Website (Frontend Project)

## 📌 Overview
This project is a **frontend fashion website** designed for an Abaya brand. It is built using **HTML and CSS** and focuses on creating a clean, elegant, and modern UI for showcasing Abaya collections.

The purpose of this project is to practice frontend development skills and improve UI/UX design understanding.

---

## 🛠️ Technologies Used
- HTML  
- CSS
-Visual Studio Code
---

## 🎯 Features
- 🧕 Elegant Abaya fashion homepage design  
- 📄 Multiple HTML pages (structured layout)  
- 🎨 Modern and responsive CSS styling  
- 🖼️ Product showcase layout  
- 🧭 Simple navigation between pages  
- 💡 Clean and minimal UI design  

---

## 📁 Project Structure
- `index.html` → Main homepage  
- `about.html / other pages` → Additional content pages  
- `style.css` → Styling and layout design  

---

## 🎯 Purpose of Project
This project was created to:
- Improve frontend development skills  
- Practice HTML structure and CSS styling  
- Learn how to design fashion-based websites  
- Build a portfolio-level UI project  

---

## ⚠️ Note
This is a **frontend-only project**. It does not include any backend functionality or real e-commerce system.
